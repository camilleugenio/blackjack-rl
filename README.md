# Reinforcement Learning on Blackjack (Gymnasium)

A reinforcement learning project implementing **Tabular Q-Learning** and 
**Deep Q-Networks (DQN)** to train an agent to play Blackjack using the 
[Gymnasium](https://gymnasium.farama.org/) `Blackjack-v1` environment.

**Authors:** Mattia Raffaele Ricciardelli · Camille Justine Gomez Eugenio  
**Course:** Machine Learning — Master's in Engineering in Computer Science  
**University:** Sapienza University of Rome, 2024/2025
